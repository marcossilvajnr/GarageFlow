# GarageFlow — Glossário de Siglas

Este documento define o significado oficial das siglas usadas na documentação DDD do GarageFlow.

## Siglas de Ordens

| Sigla | Significado | Código (EN) |
|------|-------------|-------------|
| OS | Ordem de Serviço | ServiceOrder |
| OE | Ordem de Execução | ExecutionOrder |
| OdS | Ordem de Separação | SeparationOrder |
| OdC | Ordem de Compra | PurchaseOrder |

## Siglas de Contexto e Regras

| Sigla | Significado |
|------|-------------|
| BC | Bounded Context |
| RN | Regra de Negócio |

## Siglas de Modelagem e Arquitetura

| Sigla | Significado |
|------|-------------|
| DDD | Domain-Driven Design |
| VO | Value Object |
| AR | Aggregate Root |

## Siglas de Tipos de Pessoa

| Sigla | Significado |
|------|-------------|
| PF | Pessoa Física |
| PJ | Pessoa Jurídica |
